# Папка обновлений

Сюди можна покласти файли з іншого проекту (наприклад **c:\1111**), щоб застосувати їх у **c:\NEXX LAST**.

## Що покласти для переносу з c:\1111

- `public/static/i18n.js` — оновлена локалізація (PowerStationTracker, калькулятор, UI)
- `src/components/RepairCalculator.tsx` — оновлений калькулятор
- Компоненти: PowerStationTracker, тощо (якщо окремі файли)

Після копіювання файлів у відповідні місця проекту — збережіть і запустіть `npm run build` та `npm run deploy`.
