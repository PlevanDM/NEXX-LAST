## Робочий скрипт

Скрипт `deploy-direct-api.ps1` працює з Global API Key через прямий Cloudflare Pages API (використовує curl).

### Використання:

```powershell
.\deploy-direct-api.ps1
```

Скрипт автоматично:
1. Отримує Account ID
2. Перевіряє/створює проект
3. Збирає проект
4. Створює ZIP архів
5. Завантажує через Cloudflare Pages API

## Наступні рази

Для наступних деплоїв просто виконайте:

```powershell
.\deploy-direct-api.ps1
```

Або використайте npm скрипт (після додавання до package.json):

```powershell
npm run deploy:api
```
