### Стало (после обновления)

#### 1️⃣ **Белый фон с тенью**
- Чистый белый фон шапки
- Тонкая тень `shadow-sm` для глубины
- Граница `border-b border-gray-100` для разделения

#### 2️⃣ **Логотип в контейнере**
```jsx
h('div', { className: 'bg-white rounded-xl p-2 shadow-sm border border-gray-100' },
  h('img', { 
    src: '/static/nexx-logo.png', 
    alt: 'NEXX Logo',
    className: 'h-10 md:h-12 object-contain'
  })
)
```
- Белый фон для логотипа
- Закруглённые углы `rounded-xl`
- Тонкая рамка и тень

#### 3️⃣ **Градиентный текст заголовка**
```jsx
h('h1', { 
  className: 'text-2xl md:text-3xl font-bold bg-gradient-to-r from-indigo-600 to-purple-600 bg-clip-text text-transparent' 
}, 'NEXX Database')
```
- Текст с градиентом indigo → purple
- `bg-clip-text` для клипа градиента по тексту
- `text-transparent` для прозрачности текста

#### 4️⃣ **Цветные бейджи со статистикой**
```jsx
// Устройства (indigo)
h('span', { 
  className: 'px-3 py-1.5 rounded-full bg-gradient-to-r from-indigo-50 to-purple-50 text-indigo-700 font-semibold border border-indigo-100' 
}, `📱 ${stats.total} устройств`)

// Артикулы (green)
h('span', { 
  className: 'px-3 py-1.5 rounded-full bg-gradient-to-r from-green-50 to-emerald-50 text-green-700 font-semibold border border-green-100' 
}, `🔖 ${stats.articles} артикулов`)

// Платы (blue)
h('span', { 
  className: 'px-3 py-1.5 rounded-full bg-gradient-to-r from-blue-50 to-cyan-50 text-blue-700 font-semibold border border-blue-100' 
}, `🖥️ ${stats.logicBoards} плат`)
```
- Каждый бейдж с уникальным градиентом
- Иконки эмодзи для визуализации
- Цветные рамки

#### 5️⃣ **Горизонтальный layout**
```jsx
h('div', { className: 'flex items-center justify-between flex-wrap gap-4' },
  // Logo and Title (слева)
  h('div', { className: 'flex items-center gap-3' }, ...),
  // Stats (справа)
  h('div', { className: 'flex flex-wrap gap-2 text-xs' }, ...)
)
```
- `justify-between` для разделения на левую и правую части
- `flex-wrap` для адаптивности на мобильных
- `gap-4` для отступов

#### 6️⃣ **Фоновый градиент страницы**
```jsx
h('div', { className: 'min-h-screen bg-gradient-to-br from-gray-50 via-indigo-50/30 to-purple-50/20' }, ...)
```
- Мягкий градиент от серого к индиго и фиолетовому
- Низкая opacity (30%, 20%) для ненавязчивости

### Цветовая палитра

| Элемент | Цвета | Класс Tailwind |
|---------|-------|----------------|
| Заголовок | Индиго → Фиолетовый | `from-indigo-600 to-purple-600` |
| Бейдж Устройства | Индиго → Фиолетовый | `from-indigo-50 to-purple-50` |
| Бейдж Артикулы | Зелёный → Изумрудный | `from-green-50 to-emerald-50` |
| Бейдж Платы | Синий → Голубой | `from-blue-50 to-cyan-50` |
| Фон страницы | Серый → Индиго → Фиолетовый | `from-gray-50 via-indigo-50/30 to-purple-50/20` |

### Spacing и размеры

| Элемент | Desktop | Mobile |
|---------|---------|--------|
| Логотип | 48px (`h-12`) | 40px (`h-10`) |
| Заголовок | 1.875rem (`text-3xl`) | 1.5rem (`text-2xl`) |
| Padding шапки | `py-4 px-4` | `py-4 px-4` |
| Gap между элементами | `gap-4`, `gap-3`, `gap-2` | То же |
