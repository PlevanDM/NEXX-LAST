# Чеклист проверки после деплоя

Используйте этот список после каждого деплоя на **nexx-gsm** (nexxgsm.com).

---

## 1. Перед деплоем (локально)

- [ ] `npm run build` — сборка без ошибок
- [ ] `npm run validate` — база данных валидна (предупреждения по категориям допустимы)
- [ ] При необходимости: `npm run type-check` — без критичных ошибок

---

## 2. Деплой

- [ ] Запущен деплой: `npm run deploy` или `.\deploy-2026.ps1`
- [ ] В логе нет ошибок, указан URL деплоя (например `nexx-gsm.pages.dev`)

---

## 3. Проверка деплоя (Cloudflare / URL)

- [ ] Сайт открывается: https://nexxgsm.com/
- [ ] Домен привязан к проекту **nexx-gsm** (Dashboard → Workers & Pages → nexx-gsm → Custom domains)
- [ ] При необходимости: сброс кэша Cloudflare (Purge cache) или `.\purge-cloudflare-cache.ps1`

---

## 4. Автоматические тесты

- [ ] `npm run test:functional` — чеклист функциональных проверок (SITE_URL=https://nexxgsm.com)
- [ ] `npm run test:compatibility` — совместимость
- [ ] `npm run test:security` — базовая проверка безопасности (по желанию)

Команда для функциональных тестов с продакшен-URL:
```bash
$env:SITE_URL="https://nexxgsm.com"; npm run test:functional
```

---

## 5. Ручная проверка — навигация и интерфейс

- [ ] Логотип и меню отображаются, ссылки ведут на нужные разделы
- [ ] Переключатель языка (UA/RO) работает
- [ ] Мобильное меню открывается и закрывается
- [ ] Кнопки в шапке выровнены, ничего не «плывёт»

---

## 6. Ручная проверка — калькулятор цен

- [ ] Калькулятор загружается, отображаются бренды (сетка на мобильном — 3 колонки)
- [ ] Выбор бренда → выбор модели → при наличии запчастей показывается шаг «Запчасти и цены»
- [ ] Выбор типа услуги/проблемы, расчёт цены отображается
- [ ] После расчёта — форма контакта, затем итог (шаг 7)
- [ ] Кнопка «Продолжить без модели» ведёт на шаг 4 или 5 в зависимости от данных

---

## 7. Ручная проверка — формы и контент

- [ ] Форма обратного звонка открывается, валидация телефона работает
- [ ] Картинки и иконки загружаются (Font Awesome / статика)
- [ ] Футер и ссылки (Условия, Конфиденциальность и т.д.) работают

---

## 8. Ручная проверка — Прайс Украина (если есть)

- [ ] Кнопка «Прайс Украина» в меню открывает модальное окно
- [ ] Таблица цен отображается, поиск/фильтр по продуктам работают
- [ ] Загрузка .xlsx в браузере (если включена) работает

---

## 9. Мобильная версия

- [ ] Проверка на ширине ~375px: нет горизонтального скролла
- [ ] Калькулятор и формы удобно нажимать (достаточно большие области)
- [ ] Текст читаемый без зума

---

## 10. Версия и кэш

- [ ] В DevTools → Network: скрипты грузятся с актуальными `?v=...` (например `price-calculator.js?v=4.0-20260130`)
- [ ] Жёсткое обновление (Ctrl+Shift+R) — контент обновлён, нет «старого» интерфейса
- [ ] При необходимости: Application → Clear storage → Clear site data для nexxgsm.com
