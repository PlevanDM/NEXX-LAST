## Карточка устройства

### 3 вкладки:
1. **Информация**
   - Типовые неисправности
   - Сложность и время ремонта
   - Процессор (для iPhone)

2. **Запчасти** (с переключателем UA/EU)
   - Батарея, дисплей, камеры, динамик, Taptic, клавиатура
   - Реальные цены в выбранной валюте
   - Сумма всех деталей
   
3. **Диагностика**
   - Контроллер зарядки (Tristar/Hydra/SN2800/SN2900)
   - Board Numbers

## Система цен

### Двухрегиональные цены (UA + EU)
- Украина - реальные цены в гривне
- Европа - цены Self-Repair в евро
- Сравнение экономии между регионами

### Конвертация валют:
```
1 USD ≈ 41.5 UAH
1 EUR ≈ 45.0 UAH
1 USD ≈ 0.91 EUR
```

## Калькулятор ремонта

- **Выбор региона**: Украина или Европа
- **114 устройств** с запчастями
- Виды работ: батарея, дисплей, камеры, динамик, Taptic, клавиатура, разъём
- Настройка стоимости работы
- Настройка наценки (%)
- Итоговый расчёт в выбранной валюте

## Технологии

- **Backend**: Hono на Cloudflare Workers
- **Frontend**: React 18 + Tailwind CSS (CDN)
- **Данные**: JSON файлы (real-time загрузка)

## Запуск

```bash
# Сборка
npm run build

# Локальный запуск
pm2 start ecosystem.config.cjs

# Деплой на Cloudflare
npm run deploy
```
