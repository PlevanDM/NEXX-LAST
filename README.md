## Что реализовано (v4.1)

### 7 быстрых панелей на главной странице:
| Панель | Описание | Данные |
|--------|----------|--------|
| Артикулы | Поиск по 1170 запчастям | article_search_index.json |
| Платы | 298 Logic Boards (M-series + Intel) | logic_boards_comprehensive.json |
| Цены AASP | 44 модели с офиц. ценами | official_service_prices.json |
| Ошибки | 24 iTunes + 17 Mac диагностических кодов | error_codes.json |
| Калькулятор | Расчёт стоимости ремонта | devices.json (prices) |
| База знаний | Tristar, Baseband, Touch IC, Залитие гайды | repair_knowledge.json |
| Микросхемы | IC справочник (зарядка, питание, аудио) | ic_compatibility.json |

### Главная страница
- Поиск - по модели, номеру платы, процессору
- Фильтр по категориям - iPhone, iPad, Mac
- 148 устройств в базе с полными данными

### Карточка устройства (4 вкладки):

1. **Информация**
   - Официальные цены AASP
   - Артикулы запчастей
   - Контроллер зарядки
   - Типовые неисправности
   - Сложность и время ремонта

2. **Диагностика IC**
   - Tristar U2: iPhone 6s и старее
   - Hydra: iPhone 7-14
   - USB-C: iPhone 15+
   - Симптомы и замеры

3. **Шины питания**
   - iPhone/iPad/Mac специфичные шины
   - Диодный режим и пороги КЗ

4. **Разъём (Lightning/USB-C)**
   - Распиновка
   - Мощность PD, скорость USB

### Калькулятор ремонта
- Выбор устройства из списка с ценами
- Виды работ: батарея, дисплей, камеры, динамик, зарядка
- Настройка стоимости работы ($)
- Настройка наценки (%)
- Автоматический расчёт итога

### Поиск ошибок
- iTunes/Finder ошибки (9, 14, 4013, 4014, 4005...)
- Mac диагностические коды (ADP, CNW, PFM, PPT...)
- Причина + решение
- Метка "Аппаратная проблема"

### База знаний
- Tristar/Hydra - диагностика зарядки
- Baseband - проблемы связи
- Touch IC - сенсор iPad
- Залитие - восстановление
- NAND - программирование
- Инструменты - что нужно

- **Backend**: Hono на Cloudflare Workers

- **Данные**: JSON файлы
